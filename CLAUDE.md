# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Reload / Apply Changes

```bash
# Reload i3 config in-place (no restart needed for most changes)
i3-msg reload

# Full restart (needed for changes to exec/autostart lines)
i3-msg restart
```

## Architecture

The main `config` file only sets `$mod`, `floating_modifier`, and `tiling_drag`, then delegates everything via `include` to six files under `configurations/`:

| File | Responsibility |
|---|---|
| `appearance.conf` | Font, borders, window colors, and the `bar {}` block (i3status) |
| `keybindings.conf` | All `bindsym` for apps, focus, layout, and script launchers |
| `media.conf` | Volume, brightness, and media-player key bindings |
| `workspaces.conf` | Workspace names and switch/move bindings |
| `modes.conf` | `resize`, `$system` (lock/suspend/reboot/poweroff), and `$launch` modes |
| `autostart.conf` | `exec` / `exec_always` startup programs |

## Themes

Colors are generated from theme files under `themes/<name>/colors.toml`.
The active i3 theme is rendered into `themes/current/theme/i3.conf` and included by
`configurations/appearance.conf`.

Use `scripts/theme-set <theme>` to switch themes and apply related app config for
rofi, dunst, i3status-rust, alacritty, and VS Code-compatible editors. Use
`scripts/theme-set --render-only <theme>` to regenerate the active theme without
writing external app configs or reloading services.

The built-in themes are `Ares`, `Eclipse`, `Nova`, and `Titan`.

## Scripts (`scripts/`)

Scripts are small bash helpers invoked by keybindings:

- **`settings.sh`** ‚Äî rofi menu that dispatches to other scripts
- **`theme-menu` / `theme-set`** ‚Äî select, render, and apply desktop themes
- **`bluetooth.sh`** ‚Äî lists/connects/disconnects bluetoothctl devices via rofi
- **`sink-switch.sh`** ‚Äî lists pactl sinks and switches default output via rofi
- **`volume-notify.sh`** ‚Äî wraps `pactl` for volume changes and sends a dunst notification with a progress bar
- **`brightness-notify.sh`** ‚Äî adjusts screen brightness and notifies
- **`battery-notify.sh`** ‚Äî polling daemon; shows rofi alert at ‚â§20% and ‚â§10%
- **`power-profile.sh`** ‚Äî polling daemon; writes current `power-profiles-daemon` profile to `/tmp/i3_power_profile` for i3status to read
- **`power-profile-rotate.sh`** ‚Äî cycles through performance ‚Üí balanced ‚Üí power-saver

Most rofi calls use the shared rofi config generated by the active theme. A few
older scripts may still pass explicit dmenu colors and should be migrated when
touched.

## Key Bindings Reference

`$mod` is `Mod4` (Super/Windows key). Alt and Win are swapped at the evdev level by `keyd` (`keyd/default.conf`, installed via `scripts/keyd-install`).

Notable non-obvious bindings:
- `$mod+,` ‚Üí settings menu
- `$mod+Shift+b` ‚Üí bluetooth menu
- `$mod+o` ‚Üí audio sink switcher
- `$mod+b` ‚Üí cycle power profile
- `$mod+Escape` ‚Üí system mode (lock/suspend/reboot/poweroff)
- `$mod+Shift+Space` ‚Üí launch mode (firefox/code/spotify/etc.)
